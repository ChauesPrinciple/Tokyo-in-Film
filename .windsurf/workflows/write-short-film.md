# Write Short Film Script Workflow

## When to Use
When writing or helping a student write a short film script from 3 to 30 minutes, from blank page to submission-ready draft.

## Prerequisites
- Determine the target length tier before starting (see `write-short-film-scripts` skill)
- Have access to the location(s) before locking the concept

---

## Phase 1: Concept Lock (Days 1–3)

### Day 1: The 5-Lock
Decide and commit to all of these before writing a single scene:

| Lock | Decision |
|------|----------|
| Duration | ___ minutes |
| Primary Location | _____________ |
| Secondary Location | _____________ (10+ min only) |
| Character 1 | _____________ |
| Character 2 | _____________ |
| Character 3 | _____________ (15+ min only) |
| Central Prop | _____________ |
| Dominant Emotion | _____________ |

### Day 2: The 10-Question Framework (Burnett) + McKee Story Lock
Answer every question in writing — no skipping:
1. What is the core idea?
2. Who is it about?
3. What is the genre?
4. Who/what is the antagonist?
5. What is the central question (must be answered by the end)?
6. Whose POV?
7. What happens?
8. How does it happen?
9. What is the theme?
10. How is the theme conveyed in images?

Then write the Single Sentence: "This story is about a _____ who _____ and _____ but then _____."

Then lock three McKee essentials before proceeding:
- **Controlling Idea** (VALUE + CAUSE): "_______ is achieved/lost through _______." If you cannot complete this sentence, the story is not ready.
- **Inciting Incident**: What single event radically upsets the balance of the protagonist's life? Name it. It must happen — not be implied or referred to.
- **Protagonist's Flaw** (Walley): What misconception about life does the protagonist carry into the story? The story must destroy or confirm it by the end.

### Day 3: The Beat Sheet
Use the appropriate length-specific template from the `write-short-film-scripts` skill:
- Tier 1 (3-5 min): fill in 7 beats, one every 30 seconds
- Tier 2 (5-10 min): fill in 7 beats, one every 1-2 minutes
- Tier 3 (10-20 min): fill in 4-act breakdown
- Tier 4 (20-30 min): fill in 4-act ensemble breakdown

## Phase 2: Character Engineering (Days 4–5)

### Day 4: Primary Character
Complete the Character Construction worksheet for Character 1:
- Archetype Base
- Two unique visual/behavioral details
- Objective, Mask, Wound, Breaking Point, Transformative Action
- **Flaw (antithesis)**: What does this character believe about life that is wrong? This is what the story will test.
- **Arc endpoint**: Does the character correct the flaw (positive arc), fail to correct it (tragic arc), or never recognize it (horror/irony arc)?

Map the character's visual shorthand: what three things in the first frame tell us everything about who they are?

### Day 5: Secondary and Ensemble Characters
- Complete worksheets for all remaining characters
- Map each relationship dynamic (who has power? who wants something from whom?)
- **Apply the polarization test (McKee):** Does each character contradict at least one other in attitude, value, or approach? Characters who agree cannot generate drama.
- Confirm no character exists only to deliver exposition
- For Tier 4: confirm each character has a distinct arc that intersects the A-story thematically

---

## Phase 3: Visual Storytelling (Days 6–8)

### Day 6: Location Audit
Photograph or write every corner of each location. For each space, identify:
- What natural light sources exist and when
- What physical actions are possible here that couldn't happen elsewhere
- What framing opportunities exist (doorways, reflections, depth, height)
- What sound issues might interfere with production

### Day 7: The Silent Draft
Write the entire film without a single line of dialogue.
Every beat must be communicated through action, environment, and props alone.
This is the foundation. Dialogue is added on top — never instead of this.

For each scene in the silent draft, run the **PASTO check** (Walley, from MacGowan):
- **P**reparation: what are the stakes and the character's objective entering this scene?
- **A**ttack: what does the character actively do to pursue that objective?
- **S**truggle: what specific resistance appears?
- **T**urnaround: what unexpected development changes the dynamic?
- **O**utcome: what is the new state of play — and how does it cause the next scene's Preparation?

If a scene has no Struggle or no Turnaround, it is not a scene — it is a transition. Compress it into the preceding or following scene.

Also run the **value-charge test (McKee):** What is the dominant value at stake (hope, safety, love, power, certainty) when the scene opens? Has it shifted — or shifted and reversed — by the time the scene closes? A scene that ends with the same value charge it entered accomplished nothing.

### Day 8: Motif Mapping
Identify the central visual motif (object, color, shape, gesture).
Track its appearance at each major beat and how it transforms:

| Beat | Motif Appearance | Emotional Function |
|------|-----------------|-------------------|
| Setup | | |
| Inciting Incident | | |
| Midpoint | | |
| Crisis | | |
| Climax | | |
| Resolution | | |

Also map the Central Prop journey:
- Introduction state
- Complication state
- Crisis state
- Transformation state

## Phase 4: Dialogue Integration (Days 9–10)

### Day 9: Essential Lines Only
Return to the silent draft. Add dialogue only where action cannot carry the meaning alone.
- Tier 1-2: maximum 2-3 lines per scene
- Tier 3-4: maximum 5-6 lines per scene

For every line, run the **subtext translation first (Walley)**:
- What is the character *actually* saying underneath the words?
- Characters rarely say exactly what they mean. Translate the direct version to the buried version:
  - "I'm angry you left" → "Your plant died. I kept watering it."
  - "I want you to stay" → "Traffic's terrible this time of day."
  - "You hurt me" → "I had a dog once. Ran away. Never got another."
- Can the meaning be conveyed through action or a prop instead of words at all?

**Voice differentiation (Walley):** Every character should have a distinct speech pattern — different rhythm, vocabulary, sentence length. Read each character's lines in isolation. Could these lines belong to any other character in the script? If yes, rewrite them.

### Day 10: Subtext Pass
Read the full script aloud.
- Cut 50% of remaining dialogue
- Replace any line that states what could be shown
- Check that no two characters sound identical
- Verify every line advances plot OR reveals character (ideally both)
- **Run Trottier's scene culmination check:** Does the final line of every dialogue scene land as its strongest moment? If the scene trails off, the scene is incomplete.

---

## Phase 5: Polish and Testing (Days 11–14)

### Day 11: Format and Technical
- Industry standard formatting: Courier New 12pt, margins 1.5"/1"/1"/1"
- Use `format-screenplay-docx` skill to generate the .docx
- Verify page count against target length (1 page ≈ 1 minute)
- Number all scenes
- Check all slug lines are INT./EXT. — LOCATION — TIME

### Day 12: Table Read
- Actors read aloud with the writer listening, not performing
- Note every line that sounds unnatural when spoken
- Check total timing against target length
- Mark any beats that don't land emotionally

### Day 13: Visual Test
- Storyboard 3 key scenes: opening image, midpoint, final image
- Check: does the story make sense with the dialogue muted?
- Verify the emotional beats survive without words
- Confirm the final image earns the opening image

### Day 14: Final Rewrite (Burnett's 5 Steps + Story Lock Verification)
1. **Break** — Do not touch the script for at least a few hours
2. **Interrogation** — Re-answer: core idea, genre, intended emotions, character contribution, backstory load
3. **Scene-by-scene pass** — Every scene must have purpose + end on a question that pulls into the next; verify PASTO structure holds in every scene
4. **Efficiency pass** — Cut all driving/walking transitions, duplicate information, scenes without forward momentum
5. **Dialogue polish** — Final read-aloud; cut to fewest possible words

**Story Lock Verification (run in this order before submitting):**
- Can you state the Controlling Idea (VALUE + CAUSE) in one sentence?
- Can you name the Inciting Incident? Does it appear on screen as an event, not in dialogue as backstory?
- Does the protagonist's flaw make the climax harder for them than it would be for any other person?
- Does the final image rhyme with (or deliberately invert) the opening image?
- Has every scene shifted its value charge?

## Final Checklist Before Submission

### The "So What?" Test
- [ ] Why does this story need to exist **now**?
- [ ] Why does this need to be a **film** (not a story, essay, or photo)?
- [ ] Why will the audience **remember it tomorrow**?
- [ ] What will they **feel in their body**?
