# PROFESSIONAL STORYBOARD WORKFLOW

## Complete Pipeline: Script → Thumbnails → Finished Boards → Animatic

This workflow integrates professional storyboarding curriculum with AI enhancement and traditional filmmaking skills.

---

## PHASE 1: SCRIPT BREAKDOWN (with @storyboard-artist)

### Input
- Screenplay scene
- Director's notes (if any)
- Story point (what must be conveyed)

### Process

**Step 1: Read and Mark Beats**
- Go through scene line by line
- Mark each "moment" — any discrete action or change
- Number beats sequentially

**Step 2: Identify Sequences**
- Group beats by location
- Mark sequence boundaries
- Note: New location = new establishing shot needed

**Step 3: Initial Shot Ideas**
- For each beat, jot rough camera idea
- Don't overthink — just capture instinct
- Use simple terms: "wide desert," "CU lizard," "MS digging"

**Step 4: Story Point Check**
- Review all beats
- Ensure every beat serves the story point
- Cut or combine beats that don't contribute

### Output

```
SCRIPT BREAKDOWN: [Scene Heading]

STORY POINT: [1 sentence — what audience must learn/feel]

BEATS:
1. [Line from script] → [Shot idea]
2. [Line from script] → [Shot idea]
3. [Line from script] → [Shot idea]
...

SEQUENCES IDENTIFIED:
- Sequence A: [beats X-Y] at [location]
- Sequence B: [beats Z-A] at [location]

ESTABLISHING SHOTS NEEDED:
- [List each new location requiring establishing shot]
```

## PHASE 2: THUMBNAIL PHASE (with @storyboard-artist + @ai-filmmaking-enhancement)

### Input
- Script breakdown from Phase 1
- Character designs (if available)
- Location references

### Process

**Step 1: Set Up Thumbnail Panels**
- Create simple rectangular panels (not formal storyboard template)
- No fixed dimensions — focus on composition, not precision
- Use paper, digital canvas, or simple drawing app

**Step 2: Work Through Script Sequentially**
- Draw one thumbnail per shot
- For action sequences: multiple thumbnails showing key poses
- Keep drawings rough — focus on pose, angle, composition

**Step 3: Apply Six Core Shots**
- Reference @storyboard-artist Six Core Shots
- Ask for each beat:
  - Is this establishing environment? → Establishing/Wide
  - Is this character movement? → Long shot
  - Is this hand action + face? → Medium shot
  - Is this emotion/dialogue? → Close-up
  - Is this intense detail? → Extreme close-up

**Step 4: Apply Camera Angles**
- Reference @storyboard-artist Camera Angles
- Eye level = neutral
- High angle = vulnerable
- Low angle = powerful/dominant
- Dutch = unease/wrongness

**Step 5: Apply Rule of Thirds for Narrative**
- Centered/balanced = stability, equal power
- Off-kilter/rule of thirds = conflict, tension, power imbalance
- Position characters in frame to underscore their dynamic

**Step 6: Plan Camera Moves**
- Draw arrows on thumbnails
- Pan/tracking = horizontal arrows across panels
- Tilt/jib = vertical arrows across panels
- Zoom/truck = diagonal arrows with start/end marks

**Step 7: AI Enhancement Pass**
- Reference @ai-filmmaking-enhancement
- Generate AI image prompts for complex shots
- Create mood board elements (color, texture, reference films)
- Check character consistency across thumbnails

**Step 8: Copy-Paste for Action**
- For sequential action:
  - Draw final pose first
  - Copy to previous panel
  - Erase and redraw starting pose
- Maintains proportions and spatial relationships

### Output

```
THUMBNAIL SET: [Scene Name]

ROUGH SKETCHES (describe or attach):
[Panel 1]: [Shot] - [Subject] - [Rough description]
[Panel 2]: [Shot] - [Subject] - [Rough description]
...

CAMERA MOVE NOTATION:
- Panels X-Y: [Pan/Tilt/Zoom] - [Direction]

AI IMAGE PROMPTS (for complex shots):
- Shot [N]: "[Detailed prompt for image generation]"

MOOD BOARD ELEMENTS:
- Color Palette: [Dominant] + [Accent] + [Shadow]
- Reference Films: [List 3]
- Texture: [Grain/sharpness/haze notes]

NOTES FOR FINISHED BOARDS:
[Any special instructions]
```

## PHASE 3: DIALOGUE COVERAGE (with @storyboard-artist + @staging-blocking)

### Input
- Dialogue-heavy scene from thumbnails
- Character relationship notes

### Process

**Step 1: Establish 180° Line**
- Reference @staging-blocking and @storyboard-artist
- Draw imaginary line between characters
- Mark safe zone (one side of line)
- All shots must stay in safe zone unless intentionally crossing

**Step 2: Plan Standard Dialogue Coverage**
1. **Two-Shot**: Both characters, establishes spatial relationship
2. **Singles (Close-Ups)**: One character per shot for dialogue
3. **Over-the-Shoulder (OTS)**: At least one — maintains connection

**Step 3: Plan Character Entrance (if applicable)**
- Cut to two-shot showing characters looking off-screen (motivates entrance)
- Cut to medium/long shot of new character entering
- Return to coverage pattern

**Step 4: Apply Hero Identification**
- Reference @storyboard-artist Hero Identification section
- Use POV shots to create audience identification
- Always motivate POV: show character looking off-screen first

**Step 5: Verify Screen Side**
- Character A should stay on same side (left or right) in all shots
- Character B should stay on opposite side
- If crossing 180° line is needed: use cutaway or character movement

### Output

```
DIALOGUE COVERAGE: [Scene Name]

CHARACTERS: [A] and [B]

180° LINE:
- Safe zone: [description]
- Character A screen side: [Left/Right]
- Character B screen side: [Left/Right]

SHOT SEQUENCE:
1. Two-shot (establishing)
2. Single CU - Character A
3. OTS - Over Character A's shoulder
4. Single CU - Character B
5. OTS - Over Character B's shoulder
6. [Continue or return to two-shot]

NEW CHARACTER ENTRANCE (if any):
- Shot X: Two-shot, characters looking off-screen
- Shot Y: MS/LS of new character entering
- Shot Z: Return to coverage

POV MOTIVATION:
- Shot [N]: Character A looks off-screen [direction]
- Shot [N+1]: POV of what they see
```

## PHASE 4: FINISHED STORYBOARDS (with @visual-grammar + @optics-psychology)

### Input
- Thumbnails from Phase 2
- Dialogue coverage from Phase 3 (if applicable)
- Technical specifications (aspect ratio, etc.)

### Process

**Step 1: Transfer to Storyboard Template**
- Use proper storyboard format (6-9 panels per page)
- Refine drawings from thumbnails
- Add detail but don't over-render

**Step 2: Apply Shot Grammar**
- Reference @visual-grammar
- Verify cut-off rule (no joints)
- Check lead room (space in direction of motion/gaze)
- Verify headroom (10% above head in MS/CU)
- Apply rule of thirds precisely

**Step 3: Apply Optics**
- Reference @optics-psychology
- Verify focal length matches shot size + emotional tone
- Check DOF strategy (shallow vs. deep)

**Step 4: Add Technical Information**
- Shot number
- Shot type (ELS/LS/MS/CU/ECU)
- Camera angle (eye level/high/low/Dutch)
- Camera move (if any)
- Focal length (if specified)
- Action description
- Dialogue (if any)

**Step 5: Camera Move Notation**
- Draw arrows clearly
- Mark start and end frames for zooms/trucks
- Use proper notation for diagonal pans (outside corners only)

**Step 6: Lighting Notes (Optional)**
- Reference @lighting-architecture
- Add key-to-fill ratio if relevant to mood
- Note motivated sources

### Output

```
FINISHED STORYBOARDS: [Scene Name]

[6-9 panels with refined drawings]

PANEL SPECIFICATIONS:
Panel 1:
- Shot #: 1
- Type: ELS
- Angle: Eye level
- Subject: [Description]
- Action: [Description]
- Dialogue: [If any]

Panel 2:
- Shot #: 2
- Type: MS
- Angle: Eye level
- Subject: [Description]
- Action: [Description]
- Dialogue: [If any]

[Continue for all panels...]

TECHNICAL NOTES:
- Aspect Ratio: [e.g., 2.39:1, 1.85:1]
- Camera: [If specified]
- Special Requirements: [Any]
```

## PHASE 5: ANIMATIC PREPARATION (with @editorial-grammar + @acoustic-design)

### Input
- Finished storyboards
- Dialogue recordings (if available)
- Sound effects references
- Music references

### Process

**Step 1: Scan/T photograph Boards**
- High resolution for editing software
- Consistent lighting and alignment

**Step 2: Import to Editing Software**
- Use Storyboard Pro, Premiere, or similar
- Sequence boards in order
- Match with dialogue audio

**Step 3: Time Each Shot**
- Work with director to determine shot duration
- Hold on important beats longer
- Cut faster for action sequences
- Use @editorial-grammar pacing guidelines

**Step 4: Add Temporary Sound**
- Reference @acoustic-design
- Add room tone
- Place sound effects at key moments
- Add music cues (temporary)

**Step 5: Review and Revise**
- Watch animatic multiple times
- Note pacing issues
- Mark shots that need adjustment
- Revise storyboards if needed

**Step 6: Lock Animatic**
- Once timing is approved, lock the edit
- No further timing changes without cascading impact
- Distribute to production departments

### Output

```
ANIMATIC: [Scene/Sequence Name]

DURATION: [Total runtime]

SHOT TIMING:
Shot 1: [Duration] seconds
Shot 2: [Duration] seconds
...

AUDIO ELEMENTS:
- Dialogue: [Track info]
- Room Tone: [Description]
- Sound Effects: [Key moments]
- Music: [Temporary cues]

PACING NOTES:
- Average Shot Length: [seconds]
- Rhythm: [Consistent/Accelerating/Etc.]
- Long Takes: [Which shots, why]
- Rapid Cutting: [Which sequence]

LOCKED: [Date]
APPROVED BY: [Director/Producer]

DISTRIBUTION:
- Animators: [Scene numbers]
- Background Artists: [Scene numbers]
- Prop Builders: [Prop list with scene refs]
- Editorial: [Complete sequence]
```

## PHASE 6: PRODUCTION INTEGRATION (with @production-protocol)

### Input
- Locked animatic
- Shot list from storyboards
- Production schedule

### Process

**Step 1: Create Shot List for Production**
- Reference @production-protocol
- Number shots in shooting order (not necessarily story order)
- Group by location for efficiency
- Group by lighting setup for efficiency

**Step 2: Mark Kill Shots**
- Identify shots that are absolutely mandatory
- If time runs short, these must be captured
- Prioritize based on editorial necessity

**Step 3: Create Call Sheet Entries**
- Reference @production-protocol
- List all shots per day
- Estimate setup/shoot times per shot
- Include storyboard panel references

**Step 4: Distribution to Departments**
- Storyboards to: Director, DOP, 1st AD, Gaffer, Key Grip, Art Dept, Wardrobe, Actors
- Animatic to: Editor, Sound Designer, VFX (if applicable)
- Shot list to: All production departments

### Output

```
PRODUCTION PACKAGE: [Day/Scene]

SHOTS SCHEDULED: [Number]

KILL SHOTS (Mandatory):
- Shot #: [Panel ref] - [Reason]
- Shot #: [Panel ref] - [Reason]

SHOOTING ORDER:
1. Shot [X] - [Setup time] - [Shoot time]
   [Storyboard panel reference]
2. Shot [Y] - [Setup time] - [Shoot time]
   [Storyboard panel reference]

DEPARTMENT DISTRIBUTION CHECKLIST:
- [ ] Director
- [ ] DOP
- [ ] 1st AD
- [ ] Gaffer
- [ ] Key Grip
- [ ] Art Department
- [ ] Wardrobe
- [ ] Actors
- [ ] Editorial
- [ ] Sound
- [ ] VFX

NOTES:
[Any special production requirements]
```
