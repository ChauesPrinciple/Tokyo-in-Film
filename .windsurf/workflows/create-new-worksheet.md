# Create New Worksheet Workflow

## When to Use
When you need to create a new worksheet for teaching a specific filmmaking technique or concept.

## Steps

### 1. Define Learning Objective
- What specific skill will students learn?
- How does it connect to course films?
- What's the difficulty level (beginner/intermediate/advanced)?
- How does it fit into the module sequence?

### 2. Choose Worksheet Type
- **Technique Practice**: Hands-on exercise (e.g., "Three-Shot Exercise")
- **Film Analysis**: Analyzing course films (e.g., "Visual Foreshadowing in Shoplifters")
- **Planning Document**: Pre-production tool (e.g., "Shot List Template")
- **Reflection**: Critical thinking (e.g., "Action Units Analysis")

### 3. Research and Gather Examples
- Review course films for relevant examples
- Check existing worksheets for format consistency
- Find Tokyo-specific applications
- Identify common student challenges

### 4. Structure the Worksheet
Use the standard format:
1. **Title**: Clear, descriptive
2. **Learning Objective**: What you'll master
3. **Film Reference**: Example from course film (with timestamp)
4. **Background/Theory**: Brief explanation of concept
5. **Instructions**: Step-by-step, numbered
6. **Practice Exercise**: Hands-on application
7. **Reflection Questions**: Critical thinking prompts
8. **Submission Requirements**: What to turn in

### 5. Write Content
Use `maintain-writing-voice` skill:
- **Direct**: "You will film three connected shots..."
- **Encouraging**: "This exercise builds your visual storytelling skills"
- **Specific**: "Place your subject on the right third of the frame"
- **Visual**: "Notice how Kurosawa frames the samurai against the horizon"
- **Practical**: "In Tokyo, find a location with depth (stairs, alleys, parks)"

### 6. Add Course Film Examples
- Specific scene references with timestamps
- Frame grabs if possible (describe visually)
- Explain why the example works
- Connect to student exercise

### 7. Include Tokyo Context
- Suggest accessible Tokyo locations
- Address local constraints (permissions, crowds, weather)
- Provide backup options
- Consider cultural sensitivity

### 8. Create Assessment Criteria
- Clear success metrics
- Specific, measurable outcomes
- Technical and creative components
- Process documentation requirements

### 9. Format and Polish
- Consistent heading styles
- Numbered instructions
- Bullet points for lists
- Bold key terms
- Adequate white space

### 10. Create HTML Version
- Convert from .docx to HTML
- Use course stylesheet (`style.css?v=12`)
- Maintain UTF-8 encoding for Japanese text
- Add to `/guides/` directory
- Test on mobile and desktop

### 11. Integrate into Course
- Link from relevant module page
- Add to `/free-guides.html` resource hub
- Update navigation if needed
- Add glossary terms if introducing new concepts

### 12. Test with Students
- Pilot with small group if possible
- Gather feedback on clarity
- Revise based on confusion points
- Update examples if needed

## Common Pitfalls to Avoid
- Too vague or abstract
- No specific examples
- Ignoring Tokyo context
- Unrealistic expectations
- Missing assessment criteria
- Inconsistent voice
- Poor formatting
