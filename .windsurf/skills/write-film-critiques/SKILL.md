# Write Film Critiques Skill

## Purpose
Create structured film analysis documents that teach students how to critically examine Japanese cinema through cinematography, narrative techniques, and cultural context.

## Existing Critique Documents
- `Film critiques.docx` - General film analysis templates
- `Yakuza Film Critiques.docx` - Yakuza genre-specific analysis

## Critique Framework

### Core Analysis Elements
1. **Cinematography & Visual Language**
   - Shot composition and framing
   - Camera movement and blocking
   - Lighting design and color palette
   - Mise-en-scène elements

2. **Narrative Structure**
   - Story beats and pacing
   - Character development arcs
   - Thematic elements
   - Cultural/historical context

3. **Technical Execution**
   - Editing rhythm and transitions
   - Sound design and score
   - Production design
   - Performance direction

4. **Cultural Context**
   - Japanese film traditions
   - Genre conventions
   - Historical/social commentary
   - Cross-cultural influences

## Critique Structure Template

### Introduction
- Film title, director, year, genre
- Brief synopsis (2-3 sentences)
- Thesis statement about film's significance

### Visual Analysis
- Key scenes for cinematographic study
- Shot breakdowns with timestamps
- Lighting and color analysis
- Camera technique observations

### Narrative Analysis
- Story structure examination
- Character arc tracking
- Thematic exploration
- Cultural context integration

### Technical Craft
- Editing choices and pacing
- Sound design effectiveness
- Production design details
- Performance notes

### Pedagogical Application
- What students should learn from this film
- Specific techniques to recreate
- Assignment connections
- Discussion questions

### Conclusion
- Summary of key takeaways
- Film's relevance to course objectives
- Recommended viewing context

## Writing Guidelines

### Tone and Style
- Academic but accessible
- Specific examples with timestamps
- Avoid jargon without explanation
- Connect theory to practice

### Visual References
- Include frame grabs when possible
- Cite specific scenes by timestamp
- Reference shot types by technical name
- Compare to other films when relevant

### Student-Centered Approach
- Relate analysis to student projects
- Provide actionable insights
- Connect to assignment objectives
- Encourage critical thinking

## Research Sources

### Primary Sources
- Film itself (multiple viewings)
- Director interviews
- Production notes
- Screenplay (if available)

### Secondary Sources
- Film criticism and analysis
- Cultural/historical context
- Genre studies
- Technical filmmaking resources

## Common Pitfalls to Avoid
- Plot summary instead of analysis
- Vague observations without evidence
- Ignoring cultural context
- Separating form from content
- Overlooking sound design

## Critique Formats

### Full Analysis (1500-2000 words)
For major course films requiring deep examination.

### Scene Study (500-800 words)
Focused analysis of specific sequence.

### Comparative Analysis (1000-1500 words)
Examining two films side-by-side.

### Technical Breakdown (300-500 words)
Specific technique or element focus.

## Example Analysis Structure

```
TITLE: The Geometry of Violence in Seven Samurai

INTRODUCTION
Akira Kurosawa's Seven Samurai (1954) revolutionized action cinema through 
precise spatial choreography...

VISUAL ANALYSIS
The rain battle sequence (timestamp) demonstrates Kurosawa's mastery of...
- Wide establishing shots create spatial clarity
- Diagonal compositions suggest chaos and conflict
- Weather as visual texture and narrative obstacle

ASSIGNMENT CONNECTION
Students can apply these principles in the Action Units Worksheet by...
```
