# Design Worksheets Skill

## Purpose
Create structured worksheets that guide students through specific filmmaking techniques with clear instructions, examples, and deliverables.

## Worksheet Categories

### 1. Pre-Production Worksheets
**Purpose:** Planning and preparation exercises

- **Script Breakdown** - Scene analysis and requirements
- **Storyboarding Templates** - Visual planning grids
- **Shot List Planning** - Coverage and composition planning
- **Location Scouting** - Site evaluation criteria
- **Casting & Rehearsal** - Character preparation

### 2. Production Worksheets
**Purpose:** On-set execution and technical craft

- **Three-Shot Exercise** - Essential coverage practice
- **Geometry Worksheet** - Blocking and spatial relationships
- **Camera Movement** - Motivated movement justification
- **Lighting Setup** - Light placement and rationale
- **Shoot-Day Checklist** - Production day organization

### 3. Post-Production Worksheets
**Purpose:** Editing and finishing techniques

- **Editing Rhythm** - Pacing and timing analysis
- **Color Grading** - Mood and tone through color
- **Sound Design** - Audio layering and mixing
- **VFX Planning** - Effects integration strategy

### 4. Analysis Worksheets
**Purpose:** Critical viewing and technique study

- **Action Units** - Fight choreography breakdown (Seven Samurai)
- **Visual Foreshadowing** - Spatial storytelling (Shoplifters)
- **Catharsis Analysis** - Emotional beats (Seven Samurai)
- **Genre Study** - Convention identification

### 5. Practical Skills Worksheets
**Purpose:** Hands-on technical exercises

- **Practical Effects** - Low-budget effects techniques
- **Choreography Script** - Movement notation
- **Fight Scene Structure** - Action sequence design
- **Commercial Production** - Client work workflow

## Worksheet Structure Template

### Header Section
```
WORKSHEET TITLE
Course: Tokyo in Film
Module: [Pre/Production/Post]-Production
Film Reference: [Specific film if applicable]
Time Required: [Estimated completion time]
```

### Learning Objectives
- Clear, measurable goals (3-5 bullets)
- Specific techniques to master
- Connection to course outcomes

### Materials Needed
- Equipment list
- Software requirements
- Reference materials
- Handouts or templates

### Instructions
**Step-by-step breakdown:**
1. Preparation phase
2. Execution phase
3. Review/revision phase
4. Submission requirements

### Examples
- Visual references from films
- Student work samples (if available)
- Common mistakes to avoid
- Success criteria

### Deliverables
- Specific outputs required
- Format specifications
- Submission method
- Grading criteria

### Reflection Questions
- What did you learn?
- What challenges did you face?
- How will you apply this technique?
- What would you do differently?

## Design Principles

### Clarity
- Use simple, direct language
- Number all steps
- Provide visual aids
- Define technical terms

### Scaffolding
- Break complex tasks into manageable steps
- Build on previous knowledge
- Provide examples at each stage
- Offer troubleshooting tips

### Practical Application
- Connect to real filmmaking scenarios
- Use Tokyo locations when possible
- Reference course films
- Encourage creative problem-solving

### Assessment-Ready
- Clear success criteria
- Measurable outcomes
- Self-assessment components
- Peer review opportunities

## Film-Specific Worksheets

### Seven Samurai - Action Units
**Focus:** Breaking down fight choreography into beats
- Identify action units (attack, defense, reaction)
- Map spatial relationships
- Analyze camera placement
- Study rhythm and pacing

### Shoplifters - Visual Foreshadowing
**Focus:** Using space to telegraph narrative
- Identify visual clues in mise-en-scène
- Map character relationships through blocking
- Analyze recurring visual motifs
- Apply to student projects

### Tokyo Locations - Three-Shot Exercise
**Focus:** Essential coverage at real locations
- Wide establishing shot
- Medium two-shot
- Close-up reaction
- Practice at Shibuya, Shinjuku, etc.
