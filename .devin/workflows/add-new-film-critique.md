# Add New Film Critique Workflow

## When to Use
When adding a new film to the course curriculum that requires analytical documentation.

## Steps

### 1. Research the film
- Watch the film multiple times
- Take notes on key scenes with timestamps
- Research director interviews and production notes
- Identify cultural/historical context

### 2. Create critique document
Use the **write-film-critiques** skill framework:
- Introduction with thesis
- Visual analysis (cinematography, composition, lighting)
- Narrative analysis (structure, themes, character arcs)
- Technical craft (editing, sound, production design)
- Pedagogical application (what students should learn)
- Conclusion with key takeaways

### 3. Connect to assignments
Link specific techniques to existing worksheets:
- Action choreography → Action Units Worksheet
- Visual storytelling → Visual Foreshadowing Worksheet
- Emotional beats → Catharsis Worksheet
- Spatial relationships → Geometry Worksheet

### 4. Add to module content
Update relevant module pages with:
- Film references in examples
- Scene breakdowns
- Technical analysis
- Assignment connections

### 5. Update glossary if needed
Add any new technical terms introduced by the film analysis.

### 6. Create discussion questions
Develop reflection prompts for students.

## Related Skills
- `write-film-critiques` - Use for analytical framework and structure
- `update-module-content` - Use to integrate critique into course modules
- `add-glossary-terms` - Use to add new terminology from the film
- `maintain-writing-voice` - Use to ensure consistent pedagogical tone
