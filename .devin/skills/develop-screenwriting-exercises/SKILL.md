## Exercise Types

### Scene Recreation
**Objective:** Adapt a scene from course films to Tokyo setting

**Process:**
1. Select scene from Seven Samurai, Shoplifters, etc.
2. Identify core dramatic beats
3. Translate to available Tokyo locations
4. Rewrite dialogue for student actors
5. Adjust action to match resources

**Example:**
- Original: Seven Samurai village defense
- Adapted: Yoyogi Park confrontation scene
- Changes: Scale, setting, cultural context
- Maintained: Emotional beats, character dynamics

### Original Short Film
**Objective:** Write 3-5 page original screenplay

**Requirements:**
- Set in Tokyo (use course locations)
- 2-4 characters
- Single primary location
- Clear beginning/middle/end
- Producible with student resources

**Structure:**
- Page 1: Setup and hook
- Pages 2-3: Conflict and development
- Pages 4-5: Climax and resolution

### Dialogue-Only Scene
**Objective:** Write scene using only dialogue (no action lines)

**Purpose:**
- Focus on subtext
- Practice character voice
- Understand dialogue rhythm
- Learn what dialogue can/cannot do

**Challenge:**
- Convey setting through dialogue
- Reveal character relationships
- Advance plot without action
- Create visual interest through words

### Location-First Writing
**Objective:** Write scene inspired by specific Tokyo location

**Process:**
1. Visit location (Shibuya Crossing, Suga Shrine, etc.)
2. Observe environment, people, atmosphere
3. Develop scene that uses location's unique qualities
4. Write script that couldn't work elsewhere

**Locations to Explore:**
- Shibuya Crossing - chaos, anonymity, urban energy
- Suga Shrine - quiet reflection, Your Name pilgrimage
- Shinjuku - neon nights, isolation in crowds
- Harajuku - youth culture, identity expression

### Constraint-Based Writing
**Objective:** Write within specific limitations

**Constraints:**
- One location only
- Two actors maximum
- No dialogue (silent film)
- Real-time (no time jumps)
- Single continuous shot

**Purpose:**
- Creative problem-solving
- Understanding form
- Working with limitations
- Discovering new techniques

## Screenwriting Principles for Students

### Economy
Every line must serve multiple purposes:
- Advance plot
- Reveal character
- Create atmosphere
- Build theme

### Specificity
Avoid generic descriptions:
- ❌ "A coffee shop"
- ✅ "Nomu Café, Nicolai Bergmann's Danish-style café in Aoyama"

### Producibility
Consider practical constraints:
- Available locations
- Actor capabilities
- Equipment limitations
- Time restrictions

### Cultural Sensitivity
Writing for Tokyo context:
- Research Japanese customs
- Understand social dynamics
- Respect cultural nuances
- Avoid stereotypes

## Script Format Standards

### Proper Screenplay Format
```
INT. SHIBUYA STATION - DAY

YUKI (22, art student) rushes through the crowd, 
phone pressed to ear.

                    YUKI
          (breathless)
     I'm almost there. Five minutes.

She dodges a SALARYMAN, nearly drops her portfolio.
```

### Scene Headings
- INT./EXT. - Location - TIME OF DAY
- Be specific with locations
- Use consistent naming

### Action Lines
- Present tense
- Active voice
- Visual and specific
- Brief paragraphs (3-4 lines max)

### Dialogue
- Character name centered
- Parentheticals sparingly
- Natural rhythm
- White space for readability

## Common Screenwriting Pitfalls

### Overwriting
- Too much description
- Directing on the page
- Explaining emotions
- Redundant dialogue

### Underwriting
- Vague action lines
- Generic locations
- Unclear character motivations
- Missing emotional beats

### Dialogue Issues
- On-the-nose dialogue
- Exposition dumps
- All characters sound the same
- Unrealistic speech patterns

### Structure Problems
- No clear inciting incident
- Meandering middle
- Weak climax
- Unearned resolution
